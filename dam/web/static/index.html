Migration
Script + volume backup
🚚 Full server migration in 3 steps:
1. Download zip → contains migrate.sh + dam-migrate-config.yaml
2. On source server: bash migrate.sh source → creates volumes.tar.xz alongside the script
3. Copy all 3 files to target → bash migrate.sh restore → volumes extracted + containers recreated
Bonus: Import dam-migrate-config.yaml via the Import page (accepts .zip directly)
